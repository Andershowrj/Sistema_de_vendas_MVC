# 🐾 Sistema de Vendas MVC

## 📌 Sobre o projeto
Sistema de gerenciamento de vendas desenvolvido com ASP.NET Core MVC, permitindo controle de produtos, clientes e pedidos.

## 🚀 Tecnologias
- C#
- ASP.NET Core MVC
- Entity Framework
- SQL Server

## ⚙️ Funcionalidades
- Cadastro de produtos
- Cadastro de clientes
- Registro de vendas
- Operações CRUD completas

## 🧠 O que aprendi
- Padrão MVC na prática
- Integração com banco de dados
- Adaptação entre versões do C#
- Leitura de documentação
